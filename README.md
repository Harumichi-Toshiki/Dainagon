# DainagonProject

**開発者:** 太田陽道 / 角野俊樹

## プロジェクト概要
このプロジェクトは、フロムスクラッチで「そこそこ使える」将棋AIを製作することを目的としたものです。

DL系将棋AIとして、何らかの方法で「そこそこ強い」ものを作ることを目標とします。

### Dainagonについて
このAIの特殊なところは、開発・実行をすべてDirectMLで行なっています。そのため、ほぼすべてのAI開発者にとって多少不利になります。

## 使用方法
### Python系Chunagon
- モデルのバージョンに合ったエンジンをengineフォルダからダウンロード
- vocabからvocab_v2.jsonをダウンロード
- エンジンプログラムと同じディレクトリに「model」というファイルを作り、その中にモデルを保存する。
- エンジンプログラムと同じディレクトリに「data」というファイルを作り、vocab_v2.jsonを保存する。
- batファイルを作り、以下のコードを書いて、それを将棋所やShogiGUIなどのGUIソフトにエンジンとして登録する。

```
@echo off
cd /d %~dp0
"環境に合わせたディレクトリ" -u chunagon_v(バージョンを記入)_engine.py
```

### C++系Chunagon
- モデルのバージョンに合わせたエンジンをengineフォルダからダウンロード
- 何らかの手段でコンパイルする。（ディレクトリの設定をミスっている可能性があるので、要注意）
- エンジンプログラムと同じディレクトリに「model」というフォルダを作り、その中にモデルを保存する。
- エンジンソフト（.exeファイル）をGUIソフトに登録する。
